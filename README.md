# AI Tutor Screener

## What I Built:
- I built an AI-powered tutor screening system that conducts a voice-based interview and evaluates candidates on key teaching traits like clarity, patience, and communication.

## Key Features:
- Voice-based Q&A interaction
- AI evaluation using Groq LLM
- Structured scoring system
- Final dashboard with recommendation

## Key Decisions:
- Used Groq API for fast and free inference
- Focused on voice interaction to simulate real interviews
- Prioritized core functionality over complex UI

## Tradeoffs:
- No database integration
- Limited question set
- Basic UI styling

## Challenges:
- OpenAI quota issue → switched to Groq
- JSON parsing issues → handled via extraction
- Speech recognition compatibility fixes

## Future Improvements:
- Adaptive follow-up questions
- Candidate history tracking
- Better UI/UX
